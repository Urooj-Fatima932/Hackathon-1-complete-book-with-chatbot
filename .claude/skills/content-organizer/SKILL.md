# Content Organization Skill

## When to Use This Skill
- User asks to "write a blog post" or "organize content"
- User mentions guides, tutorials, or documentation
- User wants structured output automatically

## How This Skill Works
1. Research the topic to understand context and audience
2. Create a structured outline (3-5 main sections)
3. Suggest multiple headline options (3-5 variations)
4. Draft engaging introduction (hook the reader)
5. Keep paragraphs short (max 4 sentences)
6. Highlight key points for clarity
7. Optional: Suggest visuals, examples, or code snippets if relevant

## Output Format
- Topic Summary: 2-3 sentence overview
- Target Audience: Who should read it?
- Outline: Numbered list of sections
- Headline Options: 3-5 variations
- Introduction Draft: 1-2 paragraph hook
- Key Points: 3-5 main takeaways
- Optional Suggestions: visuals, examples, or code snippets

## Example
**Input**: "Help me write a guide about using AI for hackathon projects"
**Output**:
- **Topic Summary**: How AI tools can accelerate hackathon projects for teams and individuals
- **Target Audience**: Hackathon participants and developers
- **Outline**:
  1. Why AI is useful for hackathons
  2. Choosing the right AI tools
  3. Structuring your project efficiently
  4. Common pitfalls and solutions
  5. Final tips and best practices
- **Headlines**:
  1. "Boost Your Hackathon Projects with AI in 5 Easy Steps"
  2. "AI-Powered Hacks: A Guide for Hackathon Winners"
  3. "How to Use AI to Build Better Hackathon Projects"
- **Introduction**: "Hackathons are exciting, fast-paced events where every minute counts. Using AI tools effectively can help your team prototype faster, avoid common mistakes, and impress judges. Here’s how to get started…"
- **Key Points**:
  - Use AI for rapid prototyping
  - Automate repetitive tasks
  - Validate ideas quickly
- **Optional Suggestions**:
  - Include diagrams for workflow
  - Use code snippets for common tasks
